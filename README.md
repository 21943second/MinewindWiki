# Minewind Wiki

## Contributing

A simple quickstart guide for those looking to contribute.

1. Follow this guide to get it setup https://youtu.be/zQSLq7RX8Eg
2. Ensure you have python 3.11 installed and in your path
3. Run `make setup` to install required package
4. Run `make serve` to start the live server (which can be viewed at localhost:8000)
5. Make any changes and see them reflected live
6. Add, commit, and push the changes and create a PR on my repository to get your changes added
